## ✨Features

- High Quality & Fast
- Small Website To accompany the bot
- Customizable
- Slash Commands And Context Menu Based

## Install

```sh
npm i
```

## Usage

```sh
npm run musico #with PM2
```

## Run Dev

```sh
npm run dev
```
